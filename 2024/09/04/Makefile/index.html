# Makefile 简介

​ make 和 Makefile 是 Linux 以及其他 Unix-like 系统中非常重要的工具，它们主要用于自动化编译和构建过程。通过学习和掌握 make 和 Makefile，可以极大地提高软件开发过程中的编译效率和自动化水平。

​ make 是一个命令行工具，用于解释和执行 Makefile 中的指令。当你运行 make 命令时，它会查找当前目录下的 Makefile（或 makefile，注意大小写、GNUmakefile）文件，并根据其中的指令来编译或构建项目。

​ 当你运行 make 命令时，它默认会查找名为 Makefile（大写 M）的文件。如果该文件不存在，但存在一个名为 makefile（小写 m）的文件，make 通常不会使用它，除非你明确指定了文件名。使用 make -f XXX 命令指定文件

# make 命令工作流程

  1. 读取 Makefile：make 首先读取当前目录下的 Makefile (makefile,GNUmakefile)。
  2. 确定默认目标：如果没有指定具体目标，make 会尝试构建第一个目标（通常是可执行文件）。
  3. 检查依赖性：对于每个目标，make 会检查其依赖文件是否比目标文件新。如果依赖文件更新，或者目标不存在，make 就会执行相应的规则来重建目标。
  4. 执行命令：根据规则中的命令编译或链接文件，直到所有目标都最新为止。

# Makefile 使用

​ Makefile 是一个包含一系列构建规则的文件，它告诉 make 命令如何编译和链接源代码以生成可执行文件或库。Makefile 中定义了各种目标（target）、依赖（dependency）和命令（command）
